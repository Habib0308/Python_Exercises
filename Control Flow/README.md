## Comparison Operators
```
print(true==false)
print(true!=false)
x= 5
y= 7
print(x>y)
print(x<y)
print(x<=y)
print(x>=y)
```
## Logical Operators
```
print(true!=false and x<y)
print(true!=false or x>y)
print(not true!=false)
```
## Membership Operators
```
a = 10
b = 20
list = [1, 2, 3, 4, 5 ]
if a in list:
    print("a's in the list")
else:
   print("a's not in the list")
if b not in list:
   print("b's not in the list")
else:
   print("b's in the list")
```

## Identity Operators
They are usually used to determine the type of data a certain variable contains.
```
a= "123"
b= 123
print(a == b)
print(a is b)
print(a is not b)
print(type(a) is str)
```
